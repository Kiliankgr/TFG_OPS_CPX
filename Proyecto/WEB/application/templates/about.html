Manual de Usuario

Bienvenido al sistema. Este manual tiene como objetivo guiarte en el uso de las funcionalidades principales de la plataforma.

1. Barra de Navegación

La barra superior de navegación contiene accesos a las distintas secciones del sistema:

  1. OPS_CPX: Vista principal encargada de mostrar e interactuar con las Instancias
  2. Añadir Instancia: Formulario para añadir nuevas instancias.
  3. Modo oscuro/claro: Permite cambiar el color de la Web
  4. Manual: Acceso a esta guía.

2. Pestaña OPS_CPX

En esta sección puedes visualizar y administrar todas las instancias existentes.

Las funcionalidades principales incluyen:

  1. Listado de Instancias
  2. Ordenar Instancias por fecha de creación o por nombre
  3. Visualizar detalles de cada instancia,nombre y contenido.
  4. Modificar contenido de la Instancia en cuestión.
  5. Nueva instancia, permite ir a la pestaña de creación de instancias.
  6. Ejecutar una instancia para procesar el resultado mediante el modelo.
  7. Descargar el contenido de la instancia en un archivo.
  8. Eliminar instancias que ya no sean necesarias.
  9. Modelo asociado a la Instancia si existe.
  10. Descargar el contenido del modelo en un archivo.
  11. Eliminar instancias que ya no sean necesarias.
  12. Generar gráfica con el resultado del modelo.

2.1 Visualización y obtención de datos

Aquí podremos trabajar con las Instancias creadas como ya explicamos antes. Si se desea poner a prueba una instancia solo se tendrá que seleccionar la instancia deseada y pulsar en el botón de "ejecutar", este llamará al modelo y devolverá los datos. Mientras se ejecuta el modelo la aplicación quedará bloqueada para evitar complicaciones.

Una vez finalizada la ejecución se notificará al usuario y se podrá observar la información resultante de la instancia seleccionada en la zona inferior de la pantalla, así bien también se podrá generar una gráfica con los resultados

3. Pestaña Crear Instancia

Desde esta sección puedes crear una nueva instancia de dos formas:

  1. Manual: Rellenando los campos con el nombre y contenido de la instancia directamente desde el formulario.
  2. Automática: Subiendo un archivo con los datos necesarios el contenido del fichero debe de estar en formato JSON

Una vez insertados los datos, solo será necesario confirmar, la instancia aparecerá en la pestaña OPS_CPX y estará disponible para interactuar con ella.
